## Getting started

Head to the repository on top and clone the project or download the files.

```
git clone https://github.com/GilShaashua/ng-shop.git
```

```
cd backend
npm i 
npm start
```

```
cd frontend
npm i 
npx nx serve ngshop
npx nx serve admin
```
